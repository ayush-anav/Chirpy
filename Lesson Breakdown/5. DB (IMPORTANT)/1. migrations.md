# 1. Start DB
```sh
brew services start postgresql@15 && psql postgres # postgres default DB
```

# 2. Create new DB
```sql
CREATE DATABASE chirpy;
```

# 3. Some commands
```bash 
\c chirpy # connects to DB
\dt lists table of the current db you connected to
\dn # list schema
\q # closes connection
```

---

# 4. Create folder to hold your migration
migrations 
    |
    -> schema 
        |
        -> NUMBER_NAME.SQL (001_users.sql)

##### Write your migration
```sql
-- +goose UP 

-- +goose DOWN
```

# 5. Find the URL of your DB, in the format of:
### `PROTOCOL://username:password@hostname:port/DB_name`
###### ONLY LINUX has password, so we can ignore that:
#### `psql postgres://ayushsharma@localhost:5432/chirpy`


# 6. Run migrations
```sh
goose postgres psql postgres://ayushsharma@localhost:5432/chirpy up
```
