# http.StripPrefix("WHAT TO STRIP", HANDLER)

🧠 Example
----------

http.StripPrefix("/app/", handler)

* * * * *

🔁 What it strips
-----------------

### Request comes in:

/app/index.html

### StripPrefix removes:

/app/

### What's left:

index.html

👉 This is what gets passed to the handler

* * * * *

📦 More examples
----------------

### 1

Request: /app/style.css\
Strip:   /app/\
Result:  style.css

* * * * *

### 2

Request: /app/assets/logo.png\
Strip:   /app/\
Result:  assets/logo.png

* * * * *

### 3 (important edge case)

Request: /app/\
Strip:   /app/\
Result:  "" (empty path → usually serves index.html)

* * * * *

❌ If you strip wrong prefix
---------------------------

http.StripPrefix("/", handler)

### Request:

/app/index.html

### Result:

app/index.html ❌

Now it looks for:

./app/index.html  (wrong)

* * * * *

⚡ One-line memory
-----------------

StripPrefix removes EXACTLY what you give it from the start of the URL
