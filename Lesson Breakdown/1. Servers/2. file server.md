# File-Servers

### Serves static files from the host machine.

---


# FORMAT
### Route.Handle("path", HandlerCallback)

# IN THIS CASE, ALL THE FILE in . WILL BE SERVED, EVERYTHING SERVABLE.

---

# AND ROOT will always serve index.html (because it is common convention), built in 
# the file server

---

```go 
func main() {
	router := http.NewServeMux()
	server := &http.Server{
		Handler: router,
		Addr:    ":8080",
	}

    // http.Dir() will set the path we want to serve
    // IN our FileServer, which in this case is project root
	router.Handle("/", http.FileServer(http.Dir(".")))

	err := server.ListenAndServe()
	if err != nil {
		log.Fatalf("failed to serve: %w", err)
	}
}
```
