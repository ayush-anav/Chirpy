# Custom Handlers
Right now we were using the `http.FileServer() with http.Dir()` handler to serve static directories.

#### Format:
##### `http.Handle("URL", Handler)`

---

### A http.Handler is an interface 
```go
type Handler interface {
	ServeHTTP(ResponseWriter, *Request)
}
```

## Basically any method that signs the contract </br></br> `ServeHTTP(ResponseWriter, *Request)` </br></br> is a http.Handler()

Think: **how interfaces work** AND **ALSO *logically*, to *handle http* we need to process 2 things:**

1. *Request*
2. *Response*
