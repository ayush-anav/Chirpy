# 2. server
###### we use & for:
##### 1. passing around to func (doesnt create unnecessary copy of struct)
##### 2. performance reasons 
And **set our handler as the router we made with NewServeMux()**

```go
server := &http.Server{
		Handler: router,
		Addr:    ":8080",
	}
```

# 3. server.ListenAndServe()
```go
err := server.ListenAndServe()
	if err != nil {
		log.Fatalf("failed to serve: %w", err)
	}
```

Spins up our server

---

# All together
```go
func main() {
	router := http.NewServeMux()
	server := &http.Server{
		Handler: router,
		Addr:    ":8080",
	}
	err := server.ListenAndServe()
	if err != nil {
		log.Fatalf("failed to serve: %w", err)
	}
}
```
