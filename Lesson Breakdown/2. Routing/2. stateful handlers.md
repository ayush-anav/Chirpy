# Stateful Handlers in Go — Lesson Notes

<h2>Why Stateful Handlers?</h2>

<table>
  <thead>
    <tr><th>Use Case</th><th>Example</th></tr>
  </thead>
  <tbody>
    <tr><td>Metrics</td><td>Count how many requests hit the homepage</td></tr>
    <tr><td>Database</td><td>Share an open DB connection pool</td></tr>
    <tr><td>Credentials</td><td>API keys, JWT secrets</td></tr>
    <tr><td>Caching</td><td>In-memory cache shared between handlers</td></tr>
  </tbody>
</table>

<p>Instead of globals, encapsulate shared state in a <b>struct</b> and attach handlers as <b>methods</b> on a pointer to that struct.</p>

---

<h2>The <code>apiConfig</code> Pattern</h2>

```go
type apiConfig struct {
    fileserverHits atomic.Int32
}
```

<table>
<thead>
<tr><th>Element</th><th>Why It Matters</th></tr>
</thead>
<tbody>
<tr><td>Struct holds state</td><td>Centralizes all shared, in-memory server data</td></tr>
<tr><td>Methods on <code>*apiConfig</code></td><td>Handlers gain access to state via the receiver</td></tr>
<tr><td>Pointer receiver</td><td>All handlers mutate the <i>same</i> instance</td></tr>
</tbody>
</table>

<h2><code>atomic.Int32</code> — Safe Concurrent Counter</h2>

<table>
<thead>
<tr><th>Method</th><th>Purpose</th></tr>
</thead>
<tbody>
<tr><td><code>.Add(delta)</code></td><td>Atomically adds and returns the new value</td></tr>
<tr><td><code>.Load()</code></td><td>Atomically reads the current value</td></tr>
<tr><td><code>.Store(val)</code></td><td>Atomically writes a value</td></tr>
</tbody>
</table>

<table>
<thead>
<tr><th>Approach</th><th>Concurrency Safe?</th></tr>
</thead>
<tbody>
<tr><td>Plain <code>int</code> with <code>x++</code></td><td>No — data race</td></tr>
<tr><td><code>sync.Mutex</code> + <code>int</code></td><td>Yes — but more overhead</td></tr>
<tr><td><code>atomic.Int32</code></td><td>Yes — lock-free, fast</td></tr>
</tbody>
</table>

<h2>Middleware in Go</h2>
<table>
<thead>
<tr><th>Concept</th><th>Explanation</th></tr>
</thead>
<tbody>
<tr><td>Middleware</td><td>Wraps a handler with extra logic (logging, auth, metrics)</td></tr>
<tr><td>Signature</td><td>Takes an <code>http.Handler</code>, returns an <code>http.Handler</code></td></tr>
<tr><td><code>http.HandlerFunc</code></td><td>Adapter to turn a plain function into an <code>http.Handler</code></td></tr>
<tr><td><code>next.ServeHTTP(w,r)</code></td><td>Forwards the request to the wrapped handler — must not be forgotten</td></tr>
</tbody>
</table>

```go
func (cfg *apiConfig) handlerMetrics(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "text/plain; charset=utf-8")
    w.WriteHeader(http.StatusOK)
    w.Write([]byte(fmt.Sprintf("Hits: %d", cfg.fileserverHits.Load())))
}

func (cfg *apiConfig) handlerReset(w http.ResponseWriter, r *http.Request) {
    cfg.fileserverHits.Store(0)
    w.WriteHeader(http.StatusOK)
}
```

<table>
<thead>
<tr><th>Handler</th><th>Path</th><th>Role</th></tr>
</thead>
<tbody>
<tr><td><code>handlerMetrics</code></td><td><code>/metrics</code></td><td>Reads counter, writes <code>Hits: x</code></td></tr>
<tr><td><code>handlerReset</code></td><td><code>/reset</code></td><td>Sets counter back to 0</td></tr>
<tr><td><code>middlewareMetricsInc</code></td><td>Wraps <code>/app/</code></td><td>Increments counter each hit</td></tr>
</tbody>
</table>
<h2>Wiring It Up in <code>main</code></h2>

```go
apiCfg := &apiConfig{}
mux := http.NewServeMux()

fileHandler := http.StripPrefix("/app", http.FileServer(http.Dir(".")))
mux.Handle("/app/", apiCfg.middlewareMetricsInc(fileHandler))

mux.HandleFunc("/metrics", apiCfg.handlerMetrics)
mux.HandleFunc("/reset",  apiCfg.handlerReset)
```
